# reference global

全局引用管理和维护，提供跨项目视图、垃圾回收和统计信息。

## 子命令

| 命令 | 说明 |
|:---|:---|
| `global list` | 列出所有项目及其引用的仓库 |
| `global stats` | 显示全局统计信息 |
| `global gc` | 清理过期 DB 记录和孤立缓存 |

## global list

列出所有已注册项目及其引用的仓库，按项目分组显示。

### 用法

```bash
reference global list [flags]
```

### 标志

> `--format` / `-f` 为全局标志，详见 `reference --help`。

### 示例

```bash
# 默认表格输出
reference global list

# JSON 格式
reference global list -f json
```

### 输出示例

```
+------+------------------------+------+------+-----------+
| 存在 | 项目目录               | 数量 | 类型 | 引用名    |
+------+------------------------+------+------+-----------+
|  OK  | code\cicbyte\reference |   2  | 远程 | memos-cli |
|      |                        |      | 远程 | go-git    |
+------+------------------------+------+------+-----------+
```

- **存在**：`OK` 表示项目目录存在，`!!` 表示项目目录已删除（DB 中有残留记录）

```
|   已存在    | 1      |

```

## global gc

清理过期数据，保持全局引用环境整洁。

### 用法

```bash
reference global gc [flags]
```

### 标志

| 标志 | 简写 | 说明 |
|:---|:---|:---|
| `--cache` | | 同时清理无任何项目引用的缓存目录（默认仅清理 DB 记录） |
| `--dry-run` | | 预览模式，不实际删除 |
| `--yes` | `-y` | 跳过确认提示 |

### 清理内容

| 类型 | 默认 | 说明 |
|:---|:---|:---|
| 过期 DB 记录 | 清理 | 项目目录已不存在的数据库残留记录 |
| 孤儿缓存 | 需 `--cache` | 无任何项目引用的全局缓存目录（删除后需重新 clone） |

> **知识库（wiki）不会被清理** — wiki 由 Git 管理，是持久知识资产，即使当前没有项目引用也不应删除。

### 示例

```bash
# 预览可清理项
reference global gc --dry-run

# 清理过期 DB 记录（默认行为）
reference global gc

# 同时清理孤儿缓存
reference global gc --cache

# 跳过确认直接执行
reference global gc --cache -y
```

### 输出示例

```
  [预览] 以下内容将被清理:
+----------+-----------------------------------------------+------+
| 类型     | 路径                                          | 详情 |
+----------+-----------------------------------------------+------+
| DB记录   | D:\old-project                                | 3 条引用记录 |
| 孤立缓存 | C:\Users\x\.cicbyte\reference\repos\github     |      |
+----------+-----------------------------------------------+------+

  确认执行清理? [y/N]:
```

## 相关命令

- `reference doctor` — 诊断并修复当前项目的引用健康状态
- `reference repo remove --purge` — 移除当前项目的引用并删除缓存
- `reference repo list` — 列出当前项目的引用
